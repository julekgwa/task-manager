
## MongoDB Atlas Hackathon

### Running the project

add the following variables to your env

```bash
export PASSWORD pass
export USERNAME username
export DATABASE db
```

### Clone the repo
```
git clone https://github.com/julekgwa/task-manager.git
```

```
cd task-manager

#run npm install or yarn

```
